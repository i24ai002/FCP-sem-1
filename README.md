## 📚 Lab Topics

| Lab | Topic |
|-----|-------|
| Lab 1 | Introduction to C — structure, `printf`, `scanf` |
| Lab 2 | Data types, variables, arithmetic & logical operators |
| Lab 3 & 4 | Conditional statements (`if`, `else`, `switch`) & loops (`for`, `while`, `do-while`) |
| Lab 5 & 6 | Functions, scope, and recursion |
| Lab 7 | Arrays (1D & 2D) and string manipulation |
| Lab 8 | Pointers, references, and dynamic memory |
| Lab 10 | File I/O and advanced C concepts |
| Case Study | A comprehensive project applying all semester concepts |
